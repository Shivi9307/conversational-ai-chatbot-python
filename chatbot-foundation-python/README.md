# Conversational AI Chatbot (Python)

## Overview
This project is a beginner-friendly conversational AI chatbot built using Python and Streamlit.
It maintains chat history and responds to user inputs in real time.

## Features
- Interactive chatbot UI
- Session-based chat history
- Simple rule-based responses
- Built with Python & Streamlit

## Tech Stack
- Python
- Streamlit

## How to Run
1. Clone the repository
2. Install dependencies
3. Run the Streamlit app
